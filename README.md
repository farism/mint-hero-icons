# Mint Hero Icons

[Mint lang](https://mint-lang.com) package to use [hero-icons](https://github.com/tailwindlabs/heroicons).

# [Demo](https://farism.github.io/mint-hero-icons)

The version number tracks the version number from `tailwindlabs/heroicons`.

# Usage

```mint
component Main {
  fun render {
    <{ HeroIcons:CALCULATOR }>
  }
}
```

# This package is split into three modules

`HeroIcons` - Set of `24px` standard solid icons.

`HeroIconsMini` - Set of `20px` miniature solid icons.

`HeroIconsOutline` - Set of `24px` outline icons.
